# Snake Water Gun Game 

A Python-based command-line implementation of the classic **Snake Water Gun** game, where the player competes against the computer through randomized choices.

## Overview

This project demonstrates the use of core Python concepts such as:

* User input handling
* Conditional statements
* Dictionaries
* Random number generation
* Basic game logic

It is a beginner-friendly project designed to practice problem-solving and programming fundamentals.

## Game Rules

* Snake drinks Water → **Snake wins**
* Water sinks Gun → **Water wins**
* Gun kills Snake → **Gun wins**
* Same choices result in a **Draw**

## How to Play

When prompted, enter one of the following options:

| Input | Choice |
| ----- | ------ |
| `s`   | Snake  |
| `w`   | Water  |
| `g`   | Gun    |
